# onKeyDown
Listen to the keydown event of the given key.

Refer to [onKeyStroke](https://vueuse.org/onKeyStroke) for more details.

## Usage

```js
import { onKeyDown } from '@vueuse/core'

onKeyDown('ArrowDown', (e) => {
  e.preventDefault()
})
```
