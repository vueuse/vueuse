# onKeyPressed
Listen to the keypress event of the given key.

Refer to [onKeyStroke](https://vueuse.org/onKeyStroke) for more details.

## Usage

```js
import { onKeyPressed } from '@vueuse/core'

onKeyPressed('ArrowDown', (e) => {
  e.preventDefault()
})
```
